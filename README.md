# TODO_APP
This is a full stack TODO application that uses in memory data storage to maintain todos.

### Setup Instructions

#### 1. Backend Setup

1. Navigate to the backend folder:
   ```bash
   cd week-6/6.1-todo/backend
   ```

2. Install the required dependencies:
   ```bash
   npm install
   ```
3. To run the backend code:
   ```bash
   node index.js
   ```

#### 2. Frontend Setup

1. Navigate to the frontend folder:
   ```bash
   cd week-6/6.1-todo/frontend
   ```

2. Install the required dependencies:
   ```bash
   npm install
   ```
